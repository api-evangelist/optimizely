# Optimizely GraphQL API

Optimizely Graph is a unified content query and delivery service that provides access to content across Optimizely products through a single GraphQL API. It enables flexible data retrieval, high-performance search, and dynamic content delivery for both Optimizely CMS and custom applications. Developers can explore their content model, run GraphQL queries, and power fast content delivery experiences without needing to query individual content repositories directly.
